# Student Profile Manager

BCA 3rd Semester Web Technology Project

## Technologies Used
- PHP
- HTML
- CSS
- MySQL
- XAMPP

## How to Run

1. Download the project
2. Copy folder to XAMPP -> htdocs
3. Start Apache and MySQL
4. Create database: student_project
5. Import the SQL tables
6. Open browser:
http://localhost/student_project/register.php
